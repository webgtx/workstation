### Requirements
- Ansible
- Bash
- Python3.9
- OSTREE Based OS (CoreOS, Silverblue)

### How to use
1) Run `ansible-galaxy install -r requirements.yml`
2) Execute `base.software.yml`
3) Execute `flatpak.software.yml` 
4) Exectue `shellrc.init.yml`
